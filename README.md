# VerifyNow
A Systematic Solution to the advanced to detect fake news.
This is a Flask Project Designed and Developed for the Kavach Hackathon 2023.
To Initiate 4 new ideas 
There are 4 modules mentioned which include
Clipboard Check
Extension Check
Image Check
Right News Module
